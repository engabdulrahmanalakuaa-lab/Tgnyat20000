# تقنيات سوفت Pro v3.0.0

نظام متكامل لإدارة المطاعم - تطبيق مكتبي يعمل على ويندوز بدون انترنت.

## ✨ المميزات الجديدة في الإصدار 3.0

### 🎨 واجهة جديدة كلياً
- تصميم عصري بألوان أزرق/سماوي متدرجة
- شاشة تسجيل دخول احترافية
- دعم RTL كامل وخط Tajawal العربي
- متجاوب مع شاشات اللمس
- تنبيهات Toast جميلة بدلاً من Alert

### 💰 نظام البيع المتطور (POS)
- خصم على الفاتورة (نسبة أو مبلغ)
- خصم على الصنف الواحد
- اختيار العميل من القائمة
- حفظ السلة تلقائياً في localStorage
- تنبيه المخزون المنخفض على الأصناف

### 🧾 إدارة الفواتير
- تعديل الفاتورة بعد إنشائها
- إعادة طباعة أي فاتورة سابقة
- استرجاع جزئي/كامل
- بحث متقدم (تاريخ/حالة/كابتن/عميل)
- تصدير Excel للفواتير

### 👥 إدارة كاملة
- **العملاء**: اسم/هاتف/عنوان + تقرير مشتريات
- **الموردين**: ربط مع إضافة المخزون
- **الكباتن**: تقرير أداء + رسوم بيانية
- **الطاولات**: حالة + تقرير أداء
- **المستخدمين**: صلاحيات (مدير/كاشير/مدير قسم)

### 📦 المخزون
- إضافة مخزون مع تتبع المورد والملاحظات
- جرد المخزون مع سبب التعديل
- تقرير حركة المخزون (دخول/خروج/جرد)
- تنبيهات المخزون المنخفض عند فتح الوردية
- أيقونة تحذير على المنتجات منخفضة المخزون في الـ POS

### 📊 تقارير متقدمة (مع Chart.js)
- المبيعات اليومية/الأسبوعية/الشهرية
- الأرباح حسب المنتج
- توزيع طرق الدفع
- أداء الكباتن
- أداء الطاولات
- تقرير العملاء
- المخزون والمصروفات
- تصدير Excel لكل تقرير

### 🖨️ طابعة حرارية
- إعدادات قابلة للتخصيص:
  - النوع (EPSON / STAR)
  - الواجهة (USB / Serial / Network)
  - المنفذ / العنوان
  - عرض الإيصال (58mm / 80mm)
- اختبار طباعة من شاشة الإعدادات
- توقيع المصمم محفوظ في الإيصال

### 💾 النسخ الاحتياطي
- **تلقائي** عند إغلاق التطبيق
- **تلقائي** كل 24 ساعة
- يحتفظ بآخر 7 نسخ
- نسخ احتياطي يدوي من الواجهة
- استرجاع من نسخة داخلية أو ملف خارجي

### 🔒 الاستقرار
- جميع عمليات قاعدة البيانات في `db.transaction()` (ذرية)
- تسجيل الأخطاء في مجلد `logs/`
- رسائل خطأ واضحة للمستخدم
- ترقية تلقائية للجداول القديمة (`safeAlter()`)

## 📋 المتطلبات

- ويندوز 10 أو 11 (32-bit أو 64-bit)
- ذاكرة 2 GB كحد أدنى
- مساحة 500 MB

## 🚀 التثبيت

### للمستخدم النهائي
1. حمّل ملف `Technologies-Soft-Pro-Setup-3.0.0-x64.exe` من الـ [Releases](../../releases)
2. شغّل الملف واتبع خطوات التثبيت
3. شغّل البرنامج من قائمة Start أو الأيقونة في سطح المكتب

### للمطور
```bash
# 1. استنساخ المشروع
git clone https://github.com/<user>/<repo>.git
cd technologies-soft-pro

# 2. تثبيت الاعتماديات
npm install

# 3. تشغيل التطبيق (وضع التطوير)
npm start

# 4. بناء ملف التثبيت EXE
npm run dist
# سيظهر الملف في مجلد dist/
```
